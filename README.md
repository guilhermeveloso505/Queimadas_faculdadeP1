# Queimadas: O quão perigosas podem ser?

Este projeto consiste em uma plataforma informativa dedicada à conscientização sobre o impacto das queimadas no meio ambiente e na sociedade. O site centraliza informações essenciais para educar o público sobre a diferença entre fenômenos naturais e ações humanas (antrópicas).

O objetivo principal é servir como um guia educativo que explica a origem das queimadas, seus riscos e as formas de lidar com esse problema, especialmente em contextos rurais e de preservação ambiental.
